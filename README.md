# Professional Portfolio (Next.js 14)

Production-ready portfolio website for an Electrical and Computer Engineering undergraduate specializing in AI/ML.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Features

- Research-focused sections (About, Skills, Projects, Research)
- Competitive programming highlights
- Contact form with API route (`/api/contact`)
- Dark/light mode toggle (dark default)
- Smooth scrolling and subtle motion
- SEO metadata and accessible semantic markup
